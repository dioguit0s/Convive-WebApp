Política de Privacidade

Última atualização: julho de 2026.

O Convive é um software open source e self-hosted: o código é público, e cada condomínio (ou pessoa) que o instala roda a própria cópia, no próprio servidor. Isso significa que não existe uma empresa Convive processando dados centralmente — quem controla os dados desta instância é quem administra o servidor onde ela está rodando (geralmente o síndico ou responsável técnico do condomínio).

Quais dados são armazenados

  • Dados de cadastro: nome, e-mail, apartamento e senha (armazenada com hash, nunca em texto puro).
  • Conteúdo gerado pelo uso: reservas de áreas comuns, ocorrências registradas, comunicados e notificações.
  • Foto de perfil e anexos enviados voluntariamente pelo usuário.
  • Cookies estritamente funcionais: sessão de login e "lembrar-me" (nenhum cookie de rastreamento/publicidade).

Com quem os dados são compartilhados

Por padrão, com ninguém fora da própria instância. O único envio a terceiros é o e-mail transacional (recuperação de senha, notificações) via o servidor SMTP configurado pelo administrador da instância.

Seus direitos

Para acessar, corrigir ou solicitar a exclusão dos seus dados, entre em contato com quem administra o condomínio/instância que você utiliza — normalmente pelo canal de contato desta própria aplicação.

Código aberto

Como o projeto é open source, você pode inspecionar exatamente como os dados são tratados no repositório público no GitHub.
